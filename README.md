Government Boys High School Awera Website
Overview

This project is the official school website and portal for Government Boys High School Awera.
The website is designed to provide students, teachers, parents, and visitors with easy access to educational information, downloads, registration forms, and school updates.

The portal includes a responsive design with a sidebar navigation system, gallery section, download center, registration form, Firebase login system, and multiple educational pages.

Features
🎓 Student Registration Form
🔐 Firebase Login System
📅 Date Sheet Section
📥 Download Center
👨‍🏫 Staff Section
